# Spam Email Classifier

A Machine Learning and Natural Language Processing (NLP) project developed to classify messages as Spam or Not Spam using text analysis techniques.

## Project Overview

This project uses the SMS Spam Collection Dataset to train a classification model capable of detecting spam messages. The system processes textual data, converts it into numerical features, and predicts whether a message is spam or legitimate.

## Features

- Spam message detection
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Real-time custom message prediction
- Model accuracy evaluation
- Data visualization

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Natural Language Processing (NLP)

## Machine Learning Techniques

- TF-IDF Vectorization
- Multinomial Naive Bayes Algorithm

## Dataset

SMS Spam Collection Dataset

## Project Workflow

1. Dataset loading and preprocessing
2. Text vectorization using TF-IDF
3. Model training using Naive Bayes
4. Prediction and evaluation
5. Real-time spam detection

## Output

- Spam / Not Spam prediction
- Accuracy visualization graph
